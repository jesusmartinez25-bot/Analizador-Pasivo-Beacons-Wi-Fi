📡 DESCRIPCIÓN DEL PROYECTO

OWISAM-CP es una herramienta de análisis pasivo de redes Wi-Fi enfocada en el estudio del cifrado de comunicaciones inalámbricas mediante la captura y análisis de tramas Beacon transmitidas constantemente por los routers.

El sistema permite detectar redes cercanas y evaluar su nivel de seguridad sin necesidad de conectarse a ellas, funcionando en modo completamente pasivo y no intrusivo.

🎯 Ideal para auditorías básicas, análisis académico y concienciación en seguridad inalámbrica.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🚀 FUNCIONALIDADES PRINCIPALES

📶 Escucha las señales Beacon emitidas por los puntos de acceso
🏷️ Identifica el SSID (nombre de la red)
🔐 Detecta el tipo de cifrado utilizado
🔎 Reconoce WEP
🔎 Reconoce WPA
🔎 Reconoce WPA2
🔎 Reconoce WPA3
📊 Evalúa el nivel de seguridad de cada red detectada
🛡️ Permite identificar redes vulnerables o mal configuradas

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🧠 ¿QUÉ SON LOS BEACONS WI-FI?

Los Beacons son paquetes de gestión que los puntos de acceso envían periódicamente para anunciar su presencia en el entorno inalámbrico.

Estos paquetes contienen información como:

📡 Nombre de la red (SSID)
🔐 Capacidades de seguridad
📶 Canal utilizado
🛠️ Tipo de cifrado soportado

OWISAM-CP analiza estos paquetes sin interactuar con la red, lo que garantiza:

✔️ No intrusivo
✔️ No altera el tráfico
✔️ No requiere autenticación
✔️ No realiza ataques

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔎 CLASIFICACIÓN DE SEGURIDAD

🔴 Red abierta → Muy baja seguridad
🔴 WEP → Obsoleto y vulnerable
🟠 WPA → Seguridad intermedia
🟢 WPA2 → Seguridad alta
🟢 WPA3 → Seguridad recomendada

💡 ENFOQUE DEL PROYECTO

Este proyecto está orientado al estudio del cifrado en redes inalámbricas y a la evaluación del nivel de exposición de redes Wi-Fi en un entorno determinado.

No realiza interceptación de datos privados ni acceso no autorizado a redes.
