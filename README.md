# Analizador-Pasivo-Beacons-Wi-Fi
📡 Descripción del Proyecto

OWISAM-CP es un analizador pasivo de redes Wi-Fi diseñado para estudiar el cifrado de comunicaciones inalámbricas mediante la captura y análisis de tramas Beacon transmitidas por los routers.

El sistema permite detectar redes cercanas y evaluar su nivel de seguridad sin necesidad de conectarse a ellas, funcionando en modo completamente pasivo.

🎯 Ideal para auditorías básicas, análisis académico y concienciación sobre seguridad inalámbrica.

🚀 ¿Qué hace exactamente?

El sistema:

📶 Escucha las señales Beacon transmitidas por los routers.

🏷️ Identifica el SSID (nombre de la red).

🔐 Detecta el tipo de cifrado:

WEP

WPA

WPA2

WPA3

📊 Evalúa el nivel de seguridad de la red detectada.

🛡️ Permite identificar redes vulnerables o mal configuradas.

🧠 ¿Qué son los Beacons Wi-Fi?

Los Beacons son paquetes que los puntos de acceso envían periódicamente para anunciar:

Nombre de la red

Capacidades de seguridad

Canal utilizado

Tipo de cifrado

Este proyecto analiza estos paquetes sin interactuar con la red, lo que garantiza:

✔️ No intrusivo
✔️ No altera el tráfico
✔️ No requiere autenticación
